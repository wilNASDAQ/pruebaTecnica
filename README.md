Buenos dias, buenas tardes, buenas noches, este proyecto es mi prueba técnica para Araújo & Segovia como parte del proceso de prácticas.
Este projecto se trata de una aplicación web (Frontend + Backend API REST) para gestionar productos y categorías de productos con operaciones CRUD completas.

Base de datos que utilice: PostgreSQL online en Supabase.
Backend: Java SpringBoot
Frontend: HTML5, CSS y JavaScript


Funcionalidades implementadas:
1.CRUD de Productos y Categorías, con todas las validaciones obligatorias: campos requeridos, código/nombre único, límites de caracteres, precio válido.
2.Filtrado de productos por nombre y codigo.
3.Resumen general de productos (cantidad total, promedio de precios, etc.).
4.Confirmación antes de eliminar productos.

Funcionalidades opcionales no implementadas:

esta en el codigo).
